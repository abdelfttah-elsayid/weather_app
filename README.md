A professional weather application built with **Flutter** that provides real-time weather updates with a dynamic user interface that changes its theme based on the current temperature and weather conditions.

## 🚀 Features
* **Real-time Weather Data**: Fetches data from WeatherAPI.
* **Dynamic Thermal Themes**: The app's colors (AppBar & Background) change dynamically:
    * 🧊 **Cold (e.g., 7°C)**: Cyan/Blue themes.
    * 🌿 **Moderate**: Teal/Green themes.
    * ☀️ **Warm (e.g., 30°C)**: Orange/Red themes.
* **State Management**: Uses **Flutter Bloc (Cubit)** for efficient and clean state handling.
* **Search Functionality**: Search for any city worldwide.

## 🛠️ Built With
* [Flutter](https://flutter.dev/) - UI Framework.
* [Dart](https://dart.dev/) - Programming Language.
* [Flutter Bloc](https://pub.dev/packages/flutter_bloc) - State Management.
* [Dio](https://pub.dev/packages/dio) - HTTP client for API calls.
* [WeatherAPI](https://www.weatherapi.com/) - Weather data provider.
